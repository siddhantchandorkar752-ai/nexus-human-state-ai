## 🚀 Features
- 👁️ Eye Blink Detection + Drowsiness Score
- 😤 Action Unit Based Emotion (FACS system — 7 emotions)
- 🧍 Posture + Spine Alignment Analysis
- ❤️ rPPG Heart Rate from Camera
- 🎯 Optical Flow Micro-movement Stress
- 🚨 Real-time Risk Scoring (LOW/MEDIUM/HIGH)
